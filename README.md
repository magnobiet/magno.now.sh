# magno.now.sh

Powered by [Vercel](https://vercel.com/).

## TL;DR

```bash
npm install -g now
```

### Development

```bash
git clone git@github.com:magnobiet/magno.now.sh.git && cd $_
npm install
```

### Deploy

```bash
now -n magno
now alias https://magno-$HASH.now.sh magno
```
