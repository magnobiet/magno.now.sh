# magno.now.sh

Hosted by [Zeit](https://zeit.co/).

## TL;DR

```bash
yarn global add now
```

### Development

```bash
git clone git@github.com:magnobiet/magno.now.sh.git && cd $_
yarn install
```

### Deploy

```bash
now -n magno
now alias https://magno-$HASH.now.sh magno
```
